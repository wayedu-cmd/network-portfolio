# wayedu — Network Engineering Lab Portfolio

Hands-on network design, configuration, and troubleshooting labs built in Cisco Packet Tracer, progressing from a single-office LAN to a multi-city enterprise network with branch offices.

## Why This Exists
A short paragraph aimed at whoever's reading this — what you're trying to show them, and what's worth looking at first if they only have five minutes.

> Example: I'm building toward a network engineering role and wanted a portfolio that shows real troubleshooting, not just configs that happen to work. Start with /v4 for the most advanced build, or /troubleshooting for the fault-finding write-ups.
